# 🗂️ SKILLS & MCPs INDEX

Este archivo es tu índice maestro de herramientas para ahorrar tokens y maximizar mi precisión en Antigravity. Úsalo como referencia en cada nueva conversación.

---

## 🌎 Global Skills (`~/.gemini/antigravity/skills`)

Cuidan la calidad general de todo el código generado.

- `@clean-code`: Enforce de SOLID, naming descriptivo y funciones pequeñas.
- `@testing-patterns`: Arquitectura de tests AAA y enfoque en comportamiento (no solo coverage).
- `@prompt-engineering`: Optimización de prompts para respuestas atómicas y correctas (ahorro de tokens).
- `@api-security-best-practices`: Implement secure API design patterns including authentication, authorization, input validation, rate limiting, and protection against common API vulnerabilities

## ⚡ Servidores MCP (Herramientas Externas)

Conectores de inteligencia que reducen mi necesidad de "leer todo el código".

- **`serena`**: Análisis inteligente de símbolos (clases, funciones) sin leer archivos completos.
- **`context7`**: Documentación oficial y actualizada de NestJS/Angular (evita alucinaciones).
- **`memory`**: Persistencia de contexto entre sesiones para que no "olvide" de qué trata el Gym.
- **`filesystem`**: Acceso directo y ágil a la estructura de archivos local.
- **`sequential-thinking`**: Razonamiento estructurado para resolver problemas complejos de arquitectura.
